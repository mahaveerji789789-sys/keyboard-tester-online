---

## 💡 How You Can Contribute

### 1. Report Issues
- Found a bug? Open an [Issue](../../issues)
- Keyboard key not detected correctly? Let us know!
- Browser compatibility problem? Report it!

### 2. Suggest Features
Open an issue with label `enhancement`:
- New keyboard layout support
- New language support
- UI/UX improvements

### 3. Share the Tool
Help others find this free tool:
- ⭐ Star this repository
- Share on Reddit r/MechanicalKeyboards
- Share on Twitter/X with #KeyboardTester
- Recommend in tech forums

---

## 🧪 Testing Checklist

Before reporting a bug, please test:

| Check | Description |
|---|---|
| ✅ Browser | Chrome, Firefox, Edge, Safari |
| ✅ OS | Windows, Mac, Linux |
| ✅ Keyboard Type | Mechanical, Membrane, Laptop |
| ✅ Key Type | Regular, Function, Special, Numpad |
