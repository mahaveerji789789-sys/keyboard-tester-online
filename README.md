## 🚀 What is This Tool?

**OnlineKeyTest** is a free browser-based keyboard testing tool that helps you:
- Check if your keyboard keys are working properly
- Test for stuck or dead keys
- Detect keyboard ghosting issues
- Verify key response time

Works on **PC, Laptop, Mac, Windows, Linux** — directly in your browser!
