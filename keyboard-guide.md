## 2. Types of Keyboards {#types}

### Mechanical Keyboard
- Physical switches under each key
- Long lifespan (50-100 million keystrokes)
- Tactile or clicky feedback
- Popular for gaming and programming

### Membrane Keyboard
- Rubber dome under keys
- Quieter than mechanical
- Budget-friendly
- Common in offices

### Laptop Keyboard
- Built-in, low-profile keys
- Less travel distance
- Prone to dust and liquid damage

### Gaming Keyboard
- N-key rollover (no ghosting)
- RGB lighting
- Extra macro keys
- Fast response time

## 4. Common Problems & Fixes {#problems}

### Problem: Key not working
1. Test on [test-keyboard.online](https://test-keyboard.online)
2. Clean with compressed air
3. Check USB connection
4. Update drivers
5. Try on another PC

### Problem: Multiple keys not working
1. Check USB port
2. Try different USB port
3. Test in BIOS (if works = software issue)
4. Reinstall keyboard drivers

### Problem: Keys typing wrong characters
1. Check language/keyboard layout in OS settings
2. Press Alt+Shift to toggle layout
3. Check Num Lock status

### Problem: Keyboard not detected
1. Unplug and replug USB
2. Try different USB port
3. Check Device Manager (Windows)
4. Test on another computer

## 5. Keyboard Ghosting Explained {#ghosting}

**What is ghosting?**
When you press multiple keys simultaneously, some keys don't register — this is called ghosting.

**Why does it happen?**
- Most keyboards use a matrix circuit
- Certain key combinations create "ghost" inputs
- Budget keyboards ghost at 3+ simultaneous keys

**How to test ghosting:**
👉 Go to [test-keyboard.online](https://test-keyboard.online) and press 4-6 keys at once

**Solution:**
- Get an N-key rollover (NKRO) keyboard
- Gaming keyboards usually have NKRO
