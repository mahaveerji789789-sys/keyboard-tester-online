## 📚 Keyboard Troubleshooting Guides

### Common Keyboard Problems & Solutions

**1. Key not working?**
- Test first: [test-keyboard.online](https://test-keyboard.online)
- Clean keycap and switch
- Check USB connection
- Update keyboard drivers

**2. Keyboard ghosting?**
- Test ghosting: [test-keyboard.online](https://test-keyboard.online)
- Upgrade to N-key rollover keyboard
- Avoid 6-key rollover limitations

**3. Sticky keys?**
- Remove keycap and clean underneath
- Use compressed air to remove dust
- Check OS accessibility settings
